3. Input Type

Type Description
button ปุ่มกดที่แสดงค่าของแอตทริบิวต์เป็นค่าว่างเปล่าเป็นค่าเริ่มต้น
checkbox ช่องทำเครื่องหมายอนุญาตให้เลือก / ยกเลิกการเลือกค่าเดียว
color ตัวควบคุมสำหรับการระบุสี คลิกเพื่อเปิดเครื่องมือเลือกสี
date ตัวควบคุมสำหรับการป้อนวันที่ (ปีเดือนและวันที่แบบไม่มีเวลา) เปิดตัวใช้เลือกวันที่หรือวงล้อตัวเลขสำหรับเดือนชั่วโมง ฯลฯ
datetime-local ตัวควบคุมสำหรับการป้อนวันที่และเวลาโดยไม่มี Time Zone เปิดตัวใช้เลือกวันที่หรือวงล้อตัวเลขสำหรับเดือนชั่วโมง ฯลฯ
email อินพุตรับค่าอีเมลล์ มีลักษณะเหมือนกับ text input แต่จะมีการตรวจสอบความถูกต้องของรูปแบบอีเมลล์
file ตัวควบคุมเพื่อให้ผู้ใช้เลือกไฟล์
hidden ตัวควบคุมที่ไม่ปรากฏขึ้น แต่มีการส่งค่าไปยังเซิร์ฟเวอร์
image ปุ่มส่งแบบกราฟิก - แสดงรูปภาพที่กำหนดด้วยแอตทริบิวต์ src
month ตัวควบคุมรับค่า เดือนและปี
number ตัวควบคุมสำหรับการป้อนตัวเลข แสดงสปินเนอร์และเพิ่มการตรวจสอบความถูกต้อง
password อินพุตรับข้อความแต่จะปิดบังอักขระไว้
radio ปุ่มตัวเลือกที่อนุญาตให้เลือกค่าเดียวจากหลายตัวเลือกด้วยค่าชื่อเดียวกัน
range ตัวควบคุมสำหรับการป้อนหมายเลขที่ค่าที่แน่นอนไม่สำคัญ แสดงเป็นวิดเจ็ตช่วงที่กำหนดค่าเริ่มต้นเป็นค่ากลาง ใช้ร่วมร่วม min และ max เพื่อกำหนดช่วงของค่าที่ยอมรับได้
reset ปุ่มที่รีเซ็ตเนื้อหาของแบบฟอร์มเป็นค่าเริ่มต้น
search ฟิลด์ข้อความบรรทัดเดียวสำหรับป้อนสตริงการค้นหา ตัวแบ่งบรรทัดจะถูกลบออกจากค่าที่ป้อนโดยอัตโนมัติ
submit ปุ่มกดสำหรับ submit form
tel ตัวควบคุมสำหรับการป้อนหมายเลขโทรศัพท์
text ค่าเริ่มต้น ฟิลด์ข้อความบรรทัดเดียว ตัวแบ่งบรรทัดจะถูกลบออกจากค่าที่ป้อนโดยอัตโนมัติ
time ตัวควบคุมสำหรับการป้อนค่าเวลาโดยไม่มี Time Zone
url ฟิลด์สำหรับป้อน URL ดูเหมือนว่าป้อนข้อความ แต่มีพารามิเตอร์การตรวจสอบ
week ตัวควบคุมสำหรับการป้อนวันที่ประกอบด้วยหมายเลขสัปดาห์ในปีนั้นๆ โดยไม่มี Time Zone
