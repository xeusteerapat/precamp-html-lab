1. Attribute definitions

Elements Attributes Definition
<h> class (Global Attribute) กำหนด class (Global Attribute)
id (Global Attribute) กำหนด id (Global Attribute)
<p> class (Global Attribute) กำหนด class (Global Attribute)
id (Global Attribute) กำหนด id (Global Attribute)
<div> class (Global Attribute) กำหนด class (Global Attribute)
id (Global Attribute) กำหนด id (Global Attribute)
<img> src ระบุแหล่งที่มาของรูปภาพ
alt กำหนดคำบรรยายภาพ
style กำหนดลักษณะของรูปภาพ
width กำหนดความกว้างของรูปภาพ
height กำหนดความสูงของรูปภาพ
<a> href ระบุที่หมายของ link
target กำหนดลักษณะการเปิดลิงค์
<ul> class (Global Attribute) กำหนด class (Global Attribute)
id (Global Attribute) กำหนด id (Global Attribute)
<ol class (Global Attribute) กำหนด class (Global Attribute)
id (Global Attribute) กำหนด id (Global Attribute)
<form> action กำหนดให้ form ทำอะไรต่อไป
method กำหนด HTTP verb สำหรับ form
for Specifies which form element(s) a label/calculation is bound to
onclick กำหนดว่าจะทำอย่างไรต่อเมื่อคลิกที่ form
onchange กำหนดว่าจะทำอย่างไรต่อเมื่อมีการเปลี่ยนแปลง element
<button> type กำหนดชนิดของปุ่มกด
autofocus กำหนดการ focus ของปุ่มกดเมื่อโหลดหน้าเว็บ
name กำหนดชื่อให้กับปุ่มกด
value กำหนดค่าเริ่มต้นให้กับปุ่มกด
